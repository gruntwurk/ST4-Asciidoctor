== Index Terms

[.big.red].Source.

[source,asciidoc]
..........................
A ((flow)) index term appears in both the text and the index.
A concealed index term appears only in the index.
(((concealed index term)))
(((index term,concealed)))
..........................

[.big.red].Result.

============================
A ((flow)) index term appears in both the text and the index.
